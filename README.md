* SVTRv2를 한글에 학습 시 epoch 50과 5가 큰 차이가 없음
    lr이 늘었다 줄어드는 scheduler인데, epoch이 늘어도 눈에 띄는 효과까지는 없음
    여러 평가 데이터 셋에서 어떤 경우 epoch 5가 성능이 더 좋음 (항상은 아님)
    효율 고려시 epoch 50이 전혀 필요 없다.

* lr warmup를 1.5에서 0.5로 줄이니까 좋다.
    - 원래는 전체 20epoch 학습 기준 1.5 이다.
    - 이렇게 되면 전체적인 학습 lr이 줄어서 좋지 않음
    - epoch을 5로 줄이는 만큼 warmup 시점도 줄이는 것이 좋은 것으로 보임
    - 실험 결과 줄이는 것이 미세하게 더 좋음
